### Development & Infrastructure

- **[GitHub](https://github.com/github/github-mcp-server)**
  The official GitHub MCP server. Access repositories, issues, pull requests, and file content directly from your AI assistant.

- **[Playwright](https://github.com/microsoft/playwright-mcp)**
  Browser automation for your agent. Enables reliable end-to-end testing and web interaction capabilities using the Playwright framework.

- **[Semgrep](https://semgrep.dev/docs/mcp)**
  Static analysis for security and code quality. Scan your code for vulnerabilities and enforce best practices directly within your agent's context.

- **[Linear](https://linear.app/docs/mcp)**
  Project management integration. Manage Linear issues, projects, and cycles to keep your development workflow in sync.

  Web scraping and automation platform. Access Apify's vast store of Actors to scrape data and automate web tasks.

- **[Supabase](https://supabase.com/blog/mcp-server)**
  Open source Firebase alternative. Interact with your Supabase database, run SQL queries, and manage your backend services.
